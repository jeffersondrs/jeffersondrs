## Projetos em Destaque

### 🎓 Trilhas de Aprendizagem — FTD Educação
Plataforma de e-learning com pipeline H5P/SCORM, componentes Mobirise customizados com sistema de design tokens em LESS, player de vídeo embarcado via `postMessage` e rastreamento de progresso com `guard.js` e `global-progress.js`.

### ✂️ Agendast — Sistema de Agendamento para Salões
SaaS multi-tenant com agendamento de serviços, dashboard financeiro, comissões dinâmicas por colaborador, autenticação JWT e scroll nativo para iOS.

### 🏥 Sistema de Gestão Clínica — Dra. Karla Macêdo
Prontuário eletrônico com upload de fotos via Cloudflare R2, paleta de marca personalizada e controle de consultas.

### 🛒 Empório Naturall — E-commerce
Loja virtual Next.js deployada em VPS com PM2 + Nginx + Certbot, armazenamento de imagens no R2 e autenticação via cookie JWT.
